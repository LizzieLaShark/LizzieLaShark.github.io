What is 'scope', and how does it work in javascript

In Javascript, scope relates to accessibility of defined functions and variables.
Global Scope
If you define a variable outside of a function, it can be accessed from anywhere. i.e. you can access it from inside any function.
Local Scope
If you define a variable within a function, it can only be accessed from within that function. Once you've put the closing curly bracket on your function, you will not beable to access the variable defined within it again.
Because of the fact that in javascript you can have functions within functions, scope can sometimes get a little bit confusing. A good rule of thumb is, if you've defined a variable within a function, you can't use it outside of that function. Even if that function is within another function, you can't use it outside of the function is is defined within. This means it has local scope.
